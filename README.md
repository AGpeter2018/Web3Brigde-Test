# Week 5 Web3 Test Project

This repository contains three main smart contract projects developed during Week 5. All contracts are deployed on the **Lisk Sepolia (Chain ID: 4202)** network.

## Project Summaries

### 1. SaveAsset
A decentralized savings contract that allows users to securely deposit and withdraw both Native Ether and specific ERC20 tokens.
- **Features**: Deposit/Withdraw ETH, Deposit/Withdraw ERC20, Track individual savings.
- **Deployment Addresses**:
  - **SaveAsset**: `0x250e3973dfb7da77F13c0b1216d7dD5E9BE4e87D`
  - **Mock ERC20**: `0xa93c93576B97111Afa8E6c708B02AaC948EAb20e`

### 2. Todo
A decentralized Todo list application that enables users to manage tasks on-chain.
- **Features**: Create tasks, Update titles, Mark tasks as complete with timestamps, Delete tasks.
- **Deployment Address**:
  - **Todo**: `0x4e8af530ea4e49e0c1fd25643cee04da498dc72d`

### 3. School Management System
A comprehensive system for managing students and staff within a school ecosystem, utilizing a custom token (SchoolToken) for payments and salaries.
- **Features**: Student registration (requires fees), Staff employment and salary payments, Suspension mechanisms.
- **Deployment Addresses**:
  - **SchoolManagement**: `0xcb8215374f864730a4876a29a3f530cfa45ee891`
  - **SchoolToken**: `0xd7bba16d0b49725a0b0f2ccdf7fce4210b0d9653`
